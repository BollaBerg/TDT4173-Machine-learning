# TDT 4173 - Machine learning

A repo containing files for the assignment given in TDT 4173 Machine Learning, Fall 2021.

## File overview

- The assignment is given in [introduction-notebook.ipynb](introduction-notebook.ipynb).
- The Catboost submission is calculated in [short_notebook_catboost.ipynb](short_notebook_catboost.ipynb).
- The LightGBM submission is calculated in [short_notebook_lightgbm.ipynb](short_notebook_lightgbm.ipynb).
- A correlation analysis is conducted in [correlation_analysis.ipynb](correlation_analysis.ipynb).
